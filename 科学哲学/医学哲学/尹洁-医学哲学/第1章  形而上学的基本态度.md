# 一、一元论、二元论及其医学哲学蕴含

1. 相关哲学术语：形而上学研究的是，究竟是什么构成了我们对于世界的基本态度，当我们对世界中具体事物的观点之时必定有一个关于事物本源的理解。此外，奎因的本体论承诺实际上属于元本体论领域。

2. 医学与身心问题：生物医学模式的核心恰恰是物理主义，并且往往和机械论一道，改变精神属性需要改变物理属性。但是在定义和描述疾病的时候，除生物学的基础外，患者自身对疾病的体验也构成非常重要的内容，经济、文化、社会和心理等等都被算作和生物学对立的一元，比如幻肢痛是以精神事件而非物理事件为基础。
# 二、精神疾病的分类、诊断及其意义

1. 精神医学的特殊性：精神医学中疾病的分类标准不是基于病原学基础，而是症状。有些人如福柯批评精神医学偏离了正常医学治病救人的道路，意在控制那些偏离所谓正常或规范的行为，精神医学因此变成一种控制的手段，如弗洛姆还从病态社会的角度去反驳精神医学（不过，任何类型的社会都很难避免它的社会特征让身处其中的人换上精神疾病）。对精神医学还有两点反驳，一是没有证据表明安慰剂就没有药物管用，二是很难在正常与异常状态之间化出一条清晰的界限，病理性的抑郁和重大变故的抑郁真的能区分吗。

2. 精神紊乱：医学意义上的精神紊乱就是某种机制的失败导致不能执行某种生物学的功能。理解这个概念有两点需要注意，一是比如右位心这种解剖学异常算不上紊乱，只有这种功能异常以一种负性的方式影响到生物机体整体完好；二是生物学意义的涉及是进化意义的，意在保存物种的生存和繁衍，生物学设计没有让所有人只有爱说话才能生存，也没有设计只有异性恋才能有性唤起。

3. 精神医学的分类问题：目前精神病的判断基于DSM-5，这套标准让不同精神科医师可以互通，但陷入某种诊断有效性上。比如抑郁症的判定是在九种条件中满足五个，两个同样被诊断为抑郁症的患者完全可以满足不同的条件组，很有可能他们患的是抑郁症的不同亚型甚至就不是同一种病。因为很多精神疾病症状都类似，满足疾病A的标准也可能满足疾病b的标准，最常见的是抑郁症合并焦虑症，这被称作合并症（comorbidity）。这种现象一方面会影响诊断的有效性,另一方面会给出两种不同的治疗方案给患者带来负担。目前，神经科学家一直致力于寻找精神疾病的病理生理基础，比如抑郁是血清素这类神经递质的不平衡导致的。

# 三、还原论与整体论

- 生物医学模式的主流形而上学前提就是还原论，还原论主要分为理论性的、本体论性的（如把疾病划归为基因或分子水平）的和方法论性（从心理现象到可观察行为）的三种。
# 四、实在论与反实在论

1. 实在论的内涵：实在论专注的是属性的存在是否是心智独立的。实在论主要有三种表现形式(ⅰ)直接实在论主张我们和认知对象不需要中介，所认识的就是实实在在的对象本身，而表象理论用表象作为中介稍缓和一点；(ⅱ)科学实在论承认有那些独立于心灵对其的认知或感知而存在的事物，但它也强调科学能描述这些独立实存的事物，科学反实在论者则认为科学中那些预设的实体无法对应世界中存在的事物，往往把科学当作工具；(ⅲ)建构主义除了认为不存在独立于心灵对其认知而存在的实体属性，还强调实在与发现之间的因果关系也是虚构的，我们的科学描述和解释是在某种程度上构建或刑塑“这个实在”。
2. 医学蕴涵：医学实在论可以被粗略看作科学实在论的一种，不过也有接近建构主义的所谓价值依赖型实在论，“由于有着不同的社会语境，认知能力和工具也是多变的，由此关于医学实在的描述不止一种。”
