# 一、细胞膜的结构

1.流动镶嵌模型：1972年，辛格和尼克尔森提出细胞膜的流动镶嵌模型，细胞膜由磷脂双分子层组成的流体组成，多种不同的蛋白质在磷脂双分子层中性成一种成为嵌合体或拼接体的结构，磷脂和蛋白质都可以相对运动。

2.磷脂双分子层：两层磷脂分子的亲水头部会分别对着细胞外液、胞质溶液，中间是两个分子疏水尾部组成紧密的疏水核心，少数疏水性分子可以直接经由扩散通过磷脂双层。磷脂分子并未结合，持续的随机运动会使得它们近乎自由的运动，但始终维持固有的方向。此外，大多数膜的强度相当于室温的橄榄油，具有良好的流体性和柔韧性，能够改变形状，低温会冻。动物细胞含有的类固醇能与磷脂相互作用，让其在高温下流动性下降、低温流动性上升。

3.膜上的蛋白质：包括酶、受体蛋白、识别蛋白、连接蛋白和转运蛋白。

# 二、物质如何通过细胞膜

1.被动运输：溶液中物质浓度高水的浓度就越低，物质会从浓度高的方向流向浓度低的地方，水的方向与物质的反向相反。顺梯度流动。第一是简单扩散，水、可溶性气体或脂溶性分子越过膜的脂双层。第二是协助扩散，水、离子或水溶性分子通过通道或载体跨膜。第三是渗透，水从选择透过性膜浓度高的一侧扩散至自由水浓度较低的一侧。

2.耗能运输：逆梯度流动。一是主动运输，单个小分子或离子通过跨膜蛋白进行逆浓度梯度下降的方向运输。二是内吞，质膜通过形成囊泡包裹液体、特殊液体或粒子脱离进入细胞，包括胞饮、受体介导的内吞和吞噬。三是胞吐，细胞内的膜结构将物质包裹起来，移动到细胞表面并与质膜融合，使内容物能扩散出去。![](file:///C:\Users\zhang\AppData\Local\Temp\ksohtml24084\wps29.png)

# 三、细胞之间的链接与交流

桥粒将被反复拉伸的组织连在一起，这种强有力的连接可以避免施加到这些组织上的力将组织撕裂。紧密连接使得细胞间形成一个能阻止几乎所有物质的屏障，如膀胱内细胞不让废物流回血液中、消化道的紧密连接让其免受酸、消化酶或细菌的破坏。动物细胞的间隙连接（电信号传递、平滑肌的收缩同步）和植物细胞的胞间连丝（水、营养物质和激素）让其可以直接交流。
